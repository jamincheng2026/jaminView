## 2. 工作台骨架

### 本期必须具备

- 顶部栏
- 左侧导航与资源区
- 中间画布
- 右侧属性面板
- 预览
- 发布
- 展示页

### 必须支持的动作

- 从模板进入编辑器
- 返回项目列表
- 手动保存
- 预览当前草稿
- 发布快照

### 完成标准

- 编辑器不是单页 demo，而是产品工作流中的稳定一环
- 预览与发布都能读取同一条项目链路

---

## 3. 页面级配置

### 必做功能

#### 页面

- 页面标题
- 副标题
- 页面尺寸预设
- 自定义尺寸

#### 背景

- 纯色
- 渐变
- 背景图
- 背景图遮罩
- 填充模式

#### 头部

- 头部显隐
- 头部模板
- 时间/日期
- 徽章
- 状态信息

#### 展示

- 完整适配
- 按宽度铺满
- 原始尺寸
- 居中 / 靠上

### 完成标准

- 不选组件时，右侧是页面配置，不是空面板
- 预览页与展示页同步读取页面配置
- 页面设置不会和组件设置互相污染

---

### 必做功能

- 选中

- 缩放
- 多选
- 框选
- 对齐
- 分布
- 同宽 / 同高
- 复制
- 粘贴
- 删除
- 锁定 / 解锁
- 层级调整

### 通用属性

- X / Y
- 宽 / 高
- 可见性
- 锁定
- zIndex
- 背景
- 透明度
- 边框
- 圆角
- 阴影
- 内边距

### 完成标准

- 所有组件共享统一交互逻辑

## 10. 请求配置

### P1 范围

- 请求地址
- 方法
- 刷新间隔
- 参数
- 返回映射

### 本期要求

- 先不做完整请求功能
- 但数据源模型和右侧高级区必须预留位置

### 完成标准

- 不会未来再推翻数据系统

---

## 11. 数据过滤

### P1 范围

- 条件过滤
- 排序
- Top N
- 聚合
- 截断

### 本期要求

- 先做轻量数据整形：
  - 字段映射
  - 列配置
  - 基础格式化

### 完成标准

- 数据过滤有独立产品位置，而不是散落在字段里

---

## 12. 高级事件编辑

### P1 范围

- 点击跳转
- 打开链接
- 页面跳转
- 简单联动

### 本期要求

- 在 `高级` 分组中预留结构
- 不与内容 / 数据 / 样式混在一起

### 完成标准

- 后续接事件时不需要重做面板结构

---

## 13. 装饰组件与数字翻牌

### P1 必做组件

- 边框装饰
- 装饰角标
- 装饰线
- 标签条
- 数字翻牌
- 时间类组件

### 完成标准

- 页面能做出更接近成品的大屏视觉
- 不只剩图表、地图、表格三件套

---

## 14. 实施门槛

只有下面条件同时满足，才算“这个模块可以开始写代码”：

1. 功能边界清楚
2. 右侧面板结构清楚
3. 验收标准清楚
4. 与页面级 / 数据级职责不冲突

如果做不到，就先继续补规划，不要半做半改。
