当前项目统一定位：

`JaminView` 当前第一版是一个面向业务交付场景的模板化数据大屏平台。

当前开发重点：

- 首页与产品内页按最终设计稿复现
- 模板化创建项目
- JSON / CSV / Excel 导入
- 固定画布编辑
- 预览、发布与展示页闭环

当前不作为 V1 承诺的能力：

- AI 一句话生成大屏
- NL2SQL
- 自动数据建模
- 多智能体编排
- 企业级权限与审批流

设计源：

- 当前视觉基线以 Stitch 项目为准
- 后续代码开发以前述 6 份文档和 Stitch 最终页面为依据
